# zstu-study-blindlight 🕯️

> **Not one lamp unlit but ten thousand lights the same. A spark unseen may rise, and set the world aflame.**

## 📖 项目简介
本项目是 **浙江理工大学 (ZSTU)** 计算机相关专业的学习资料整理仓库。

希望能为大家的学习带来些许帮助，若学大家皆好学用，扎实基础，攻克难点。也就是所谓的“盲灯”计划——希望能点亮未知的领域。

仓库内容包含：
- ✅ **课程作业**：各学期实验代码、大作业源码
- 📝 **复习资料**：期末复习重点、笔记整理
- 🛠️ **项目实战**：Web 开发、人工智能、软件工程等课程设计
